Desarrollo de una pagina web con HTML

Que es HTML?

HTML (por sus siglas en ingles, HyperText Markup Language), es un lenguaje de marcado que nos permite hacer la estructura de una pagina web. Tenemos que aclarar que HTML no es un lenguaje de programacion, pues carece de mecanismos para ejecutar calculos, repeticiones o condiciones, lo que si es, es un lenguaje de marcado. Es un lenguaje que le indicara al navegador web que tipo de elemento es el que esta en la estructura de una pagina web.

Lo primero tenemos que crear la estructura del HTML que contiene :

  • Una etiqueta HTML
  • Dentro de la etiqueta HTML la etiqueta HEAD
  • La etiqueta HEAD puede contener etiquetas como TITLE , LINK Y SCRIPT:
    • SCRIPT: Llamamos al archivo externo de JavaScript, es decir, no estará incluido en el HTML.
    • LINK: Llamamos al archivo externo de CSS, es decir, no estará incluido en el HTML.
    • TITLE: Título de la página.
  • La etiqueta BODY: Dentro de esta etiqueta se pueden crear diferentes etiquetas para agregar elementos a nuestra página web. Los más utilizados son:
    • H1/2/3/4/5/6 para títulos
    • P para párrafos
    • Form para formularios
    • Button para botones
    • Label e Input para recoger datos
    • Img para imagenes
    • Ul y Li para crear una lista con viñetas
    • Ol y Li para crear una lista enumerada
    • Table - Tr - Th - Td para añadir una tabla
